# ❓ 문제

> [백준 28702 - "FizzBuzz"](https://www.acmicpc.net/problem/28702)

## 🎯 난이도

> Bronze 🟤1

## 🧠 풀이

### 1. 내 풀이 (문자열 + 수학)

#### 알고리즘

- `String`, `Mathematics`

#### 설명

`문자열`을 사용한 숫자 판별로 푼 방식이다.

`3의 배수`와 `5의 배수`를 제외하고 숫자가 나온다면, 3개의 출력이 나왔을 때 **<u>무조건 하나 이상은 정수가 나올 수 밖에 없다.</u>** 이런 규칙을 이용해서, 입력에 대해 각각의 입력 `문자열`이 정수인지 판별하는 함수를 만들어서 해결했다.

문자열을 정수 판별하는 반복문과 입력 반복문 밖에 없으므로 사실상 `O(1)`의 시간 복잡도이다.
{: .notice--info}

#### 코드

<details>
<summary style = "color:lightblue">내 풀이 코드</summary>
<div markdown="1">

```cpp
#include <iostream>
#include <string>
#include <cctype>

using namespace std;

// 문자열이 숫자인지 판별하는 함수
bool IsNumber(const string& strInput);

int main()
{
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    string arrStr[3]{};
    for(int i = 0; i < 3; ++i)
    {
        cin >> arrStr[i];
    }

    int iVal{}, iIndex{};
    for(int i = 0; i < 3; ++i)
    {
        if(IsNumber(arrStr[i]))
        {
            // 숫자 찾으면 iVal에 저장, iIndex에 인덱스 저장
            iVal = stoi(arrStr[i]);
            iIndex = i;
        }
    }

    // 다음에 올 숫자 계산
    int iNext{ iVal + 3 - iIndex };

    // 3의 배수 && 5의 배수
    if(iNext % 15 == 0)
    {
        cout << "FizzBuzz";
    }
    // 3의 배수
    else if(iNext % 3 == 0)
    {
        cout << "Fizz";
    }
    // 5의 배수
    else if(iNext % 5 == 0)
    {
        cout << "Buzz";
    }
    // 둘 다 X
    else
    {
        cout << iNext;
    }

    return 0;
}

bool IsNumber(const string& strInput)
{
    for(const char ch : strInput)
    {
        // isdigit으로 문자가 숫자인지 판별 -> 문자 검사이므로 안전하게 unsigned char로 캐스팅
        if(!isdigit(static_cast<unsigned char>(ch)))
        {
            return false;
        }
    }

    // 빈 문자열이면 false 리턴
    return !strInput.empty();
}
```
