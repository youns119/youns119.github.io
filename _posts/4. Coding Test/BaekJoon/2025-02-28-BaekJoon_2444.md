# ❓ 문제

> [백준 2444 - "별 찍기 - 7"](https://www.acmicpc.net/problem/2444)

## 🎯 난이도

> Bronze 🟤3

## 🧠 풀이

### 1. 내 풀이 (구현)

#### - 알고리즘

- `Implementation`

#### - 설명

단순 반복문으로 하는 `구현` 문제이다.

여러 `별 찍기` 시리즈의 문제 중 하나로, 공백과 별의 갯수를 적절한 공식으로 구하고 반복문으로 출력을 하는 간단한 풀이이다. 

이중 반복문이므로, 시간 복잡도는 `O(N²)`이다.
{: .notice--info}

#### - 코드

<details>
<summary style = "color:lightblue">내 풀이 코드</summary>
<div markdown="1">

```cpp
#include <iostream>

#include <cmath>


using namespace std;

int main()
{
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int iN{};
    cin >> iN;

    for(int i = 1; i <= 2 * iN - 1; ++i)
    {
        int iSpace{ abs(iN - i) };
        int iStar{ 2 * (iN - iSpace) - 1 };

        for(int i = 0; i < iSpace; ++i)
        {
            cout << ' ';
        }
        for(int i = 0; i < iStar; ++i)
        {
            cout << '*';
        }

        cout << '\n';
    }

    return 0;
}
```
