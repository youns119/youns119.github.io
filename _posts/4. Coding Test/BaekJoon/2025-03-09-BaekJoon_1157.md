# ❓ 문제

> [백준 1157 - "단어 공부"](https://www.acmicpc.net/problem/1157)

## 🎯 난이도

> Bronze 🟤1

## 🧠 풀이

### 1. 내 풀이 (구현)

#### - 알고리즘

- `Implementation`, `string`

#### - 설명

`문자열`을 활용한 구현 풀이이다.

`문자열`을 입력 받고 각 문자들을 순회하면서, **<u>대소문자에 상관없이 빈도수를 갱신해가며 '최빈도'인 알파벳을 대문자로 출력</u>**하도록 로직을 구현했다.

대소문자에 상관없이 인식하도록 `toupper` 함수로 모든 문자를 대문자로 변환했으며, 이전 반복문에서 구한 `iMax (최빈도)`를 기준으로 같은 최빈도 알파벳이 있으면 `?`를 출력, 아니라면 해당 알파벳을 출력하도록 했다.

`sort`를 사용하거나 `max_element` 등의 함수를 사용하는 방법도 있겠지만, 나에겐 이 풀이가 가장 효율적이라고 판단되었다.

처음 `O(N)` 반복문을 제외한 `toupper`은 `O(1)`, 두 번째 반복문 또한 상수 시간으로 `O(1)`이므로, 전체적인 시간 복잡도는 `O(N)`이다.
{: .notice--info}

#### - 코드

<details>
<summary style = "color:lightblue">내 풀이 코드</summary>
<div markdown="1">

```cpp
#include <iostream>

#include <string>

#include <cctype>

#include <algorithm>


using namespace std;

int main()
{
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    string strInput{};
    cin >> strInput;

    int arrAlphabet[26]{};  // 각 알파벳 개수 배열
    int iMax{}; // 최빈도 변수

    for(char chInput : strInput)
    {
        // 현재 문자대문자로 변환
        chInput = static_cast<unsigned char>(toupper(chInput));
        
        int iInput{ static_cast<int>(chInput - 'A') };

        // 해당 알파벳의 빈도 증가, 최빈도 변수 갱신
        ++arrAlphabet[iInput];
        iMax = max(iMax, arrAlphabet[iInput]);
    }

    int iIndex{ -1 };   // 최빈도 알파벳의 인덱스 변수
    for(int i = 0; i < 26; ++i)
    {
        // 최빈도와 같은 빈도를 가진 알파벳인 경우
        if(arrAlphabet[i] == iMax)
        {
            // 이미 이전에 최빈도가 나온 경우
            if(iIndex != -1)
            {
                cout << '?';
                return 0;
            }

            // 인덱스 갱신
            iIndex = i;
        }
    }

    // 대문자로 출력
    cout << static_cast<char>(iIndex + 'A');

    return 0;
}
```

# 💭 후기

문자열의 `대소문자` 판별, `최빈도` 및 `중복 원소`의 판별 등의 로직은 코테를 하면서 정말 많이 나오고, 익혀두면 아주 좋으므로 쉬운 난이도의 문제라도 정리해보고자 한다.
