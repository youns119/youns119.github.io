# ❓ 문제

> <https://www.acmicpc.net/problem/10951>

## 🎯 난이도

> Bronze 🟤5

## 🧠 풀이

### 1. 내 풀이

#### 알고리즘

- `Mathematics`, `Implementation`

#### 설명

`Bronze 🟤5`인 문제를 굳이 풀이를 올리는 이유는 좀 특별하다. 이 문제의 조건이 좀 특이하기 때문이다.

이 문제의 입력에는 입력 횟수를 제한하거나, 특정 조건에서 입력이 끝난다는 식의 조건이 존재하지 않는다. 백준의 온라인 저지 환경에서 주어지는 테스트 입력의 끝, **<u>즉 `EOF(End Of File)`을 확인해야 한다.</u>**

```yaml
1 1
2 3
3 4
(여기서 EOF)
```

이런 식으로 계속해서 두 정수를 받다가 더 이상 읽을 데이터가 없으면 `EOF` 상태가 되고, 이 때 루프를 종료하면 된다. 이는 다음과 같이 가능하다

```yaml
// cin으로 입력 받기
while (cin >> A >> B)

// scanf로 입력 받기
while (scanf("%d %d", &A, &B) != EOF)
```

`cin`의 경우로 설명하자면, `cin`으로 입력을 받으면 연쇄 연산을 위해 `cin`객체 자체를 반환한다. 입력에 성공하면 true, 실패하면 false로 `cin` 객체의 상태가 저장이 되고 자체적인 `bool` 연산자 오버로딩에 의해 `while`문 내부의 조건식이 만족되는 것이다.

#### 코드

<details>
<summary style = "color:lightblue">내 풀이 코드</summary>
<div markdown="1">       

```cpp
#include <iostream>

using namespace std;

int main()
{
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int iA{}, iB{};

    while(cin >> iA >> iB)
    {
        cout << iA + iB << "\n";
    }

    return 0;
}
```

</div>
</details>
